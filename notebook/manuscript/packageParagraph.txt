For all analysis we used R (v.4.4.2) [@base], and R Studio (v.2024.12.0+467) [@rstudio]. For analysis of animal movement data we used amt (v.0.2.2.0) [@amt], ctmm (v.1.2.0) [@ctmm], and move (v.4.2.6) [@move]. For general data manipulation we used glue (v.1.8.0) [@glue], sjmisc (v.2.8.10) [@sjmisc], tidyverse (v.2.0.0) [@tidyverse], and units (v.0.8.5) [@units]. For project and code management we used here (v.1.0.1) [@here], tarchetypes (v.0.11.0) [@tarchetypes], and targets (v.1.9.0) [@targets]. For visualisation we used the following as expansions from the tidyverse suite of packages: ggdist (v.3.3.2) [@ggdist2024a;@ggdist2024b], ggridges (v.0.5.6) [@ggridges], ggtext (v.0.1.2) [@ggtext], patchwork (v.1.3.0) [@patchwork], and scales (v.1.3.0) [@scales]. Other packages we used were boot (v.1.3.31) [@boot2024;@boot1997], circular (v.0.5.1) [@circular], doParallel (v.1.0.17) [@doParallel], foreach (v.1.5.2) [@foreach], knitr (v.1.49) [@knitr2024;@knitr2015;@knitr2014], and usethis (v.3.0.0) [@usethis]. To generate typeset outputs we used bookdown (v.0.42) [@bookdown2025;@bookdown2016], and rmarkdown (v.2.29) [@rmarkdown2024;@rmarkdown2018;@rmarkdown2020]. To manipulate and manage spatial data we used gdistance (v.1.6.4) [@gdistance], raster (v.3.6.30) [@raster], sf (v.1.0.19) [@sf2023;@sf2018], sp (v.2.1.4) [@sp2005;@sp2013], terra (v.1.7.83) [@terra], and tidyterra (v.0.6.1) [@R-tidyterra]. To run models and explore model outputs we used effects (v.4.2.2) [@effects2019;@effects2018;@effects2003;@effects2009], INLA (v.24.6.27) [@INLA2013b;@INLA2015d], lme4 (v.1.1.35.5) [@lme4], and performance (v.0.12.4) [@performance].
